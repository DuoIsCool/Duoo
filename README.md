# SmartEyeWeb
智“眼”守护，让“爱”常伴--失能老人智能监护Web管理系统

## 开始

1. 使用以下命令安装依赖库
```
pip3 install -r requirements.txt
```

2. 运行app.py文件

## 项目结构

- **blueprints** 存放蓝图
- **static** 静态文件
- **templates** 模板文件
- **app.py** 主文件
- **extensions.py** 扩展类实例化
- **factory.py** 工厂
- **models.py** 模型文件
- **settings.py** 配置文件
